You can use **GSAP** + **ScrollTrigger** in React client components for rich UI polish.

import gsap from "gsap";
import ScrollTrigger from "gsap/ScrollTrigger";

gsap.registerPlugin(ScrollTrigger);

// Animate on scroll
gsap.to(".box", {
y: 100,
scrollTrigger: ".box"
});


- Only import GSAP in `"use client"` components!
- Compose Timeline for smooth entry/exit effects.
